Key Rules And Formulas

  • sin(theta) = opposite / hypotenuse
  • cos(theta) = adjacent / hypotenuse
  • tan(theta) = opposite / adjacent
  • theta = sin^-1, cos^-1, tan^-1
  • Use Pythagoras first if needed

Common Mistakes

  • Using radians mode by accident
  • Picking wrong trig ratio
  • Rounding too early
