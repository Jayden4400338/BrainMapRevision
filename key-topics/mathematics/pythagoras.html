Key Rules And Formulas

  • a^2 + b^2 = c^2
  • c = sqrt(a^2 + b^2)
  • a = sqrt(c^2 - b^2)
  • Only for right-angled triangles
  • If x^2 + y^2 = z^2 then right-angled

Worked Examples

Find c for legs 6 and 8

  1. Square then add then square root

Answer: 10

Find short side with c=13 and b=5

  1. Compute sqrt(13^2 - 5^2)

Answer: 12

Test 7,24,25

  1. Check if 7^2 + 24^2 equals 25^2

Answer: Right-angled

Common Mistakes

  • Using theorem for non-right triangles
  • Choosing wrong hypotenuse
  • Forgetting to square first
  • Rounding before final step

Revision Checklist
