Key Rules And Formulas

  • Mean = total / number of values
  • Range = largest - smallest
  • Estimated mean = sum(fx) / sum(f)
  • Frequency density = frequency / class width
  • Cumulative frequency is running total

Common Mistakes

  • Using grouped methods on raw data
  • Forgetting to order data before median
  • Mixing frequency with frequency density
  • Comparing sets using only one statistic
